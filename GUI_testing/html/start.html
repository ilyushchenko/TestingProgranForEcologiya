Программа расчета нормированных показателей выбросов вредных веществ в атмосферу и в водную среду для экономической оценки регулирования природопользования.

(электронный учебник)




Допущено УМО по образованию в области кораблестроения и океанотехники для студентов высших учебных заведений, обучающихся по направлениям подготовки бакалавров, магистров 080500 «Менеджмент» и 080100 «Экономика», 280700 «Техносферная безопасность», 656600 «Защита окружающей среды».





 Чихонадских Е.А., Маликов А.Т.
  
 

В программе представлен расчет нормированных показателей выбросов вредных веществ в атмосферу и в водную среду для экономической оценки регулирования природопользования, на основе решения эколого-экономических проблем, возникающих при использовании природных ресурсов.

Приведены механизмы экологического регулирования природопользования.

Электронный учебник предназначен для студентов и специалистов, для использования в практической профессиональной деятельности.
